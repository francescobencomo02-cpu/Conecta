# 🌐 Conecta - Red Social

Bienvenido a **Conecta**, una red social moderna construida con HTML, CSS, JavaScript y localStorage. ¡Conecta con el mundo!

## ✨ Características

- 🔐 **Autenticación**: Registro e inicio de sesión de usuarios
- 📝 **Publicaciones**: Crea y comparte publicaciones con la comunidad
- 💬 **Mensajes Directos**: Chatea con otros usuarios en tiempo real
- 👤 **Perfil de Usuario**: Personaliza tu perfil con foto de portada y bio
- 🔔 **Notificaciones**: Recibe notificaciones de interacciones
- ❤️ **Reacciones**: Dale "Me gusta" a las publicaciones
- 👥 **Seguimiento**: Sigue a otros usuarios y crea tu red
- 🔍 **Búsqueda**: Busca publicaciones y usuarios
- 📱 **Responsive**: Diseño adaptable a cualquier dispositivo

## 🚀 Cómo Empezar

### Requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- No requiere instalación de dependencias

### Instalación

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/francescobencomo02-cpu/Conecta.git
   cd Conecta
   ```

2. **Abre el proyecto**:
   - Abre `index.html` en tu navegador

3. **Crea una cuenta**:
   - Haz clic en "Crear Cuenta"
   - Completa el formulario de registro
   - ¡Listo! Ya puedes empezar a usar Conecta
